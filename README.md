# 💣 數學拆彈專家 (Math Bomb Defuser)

這是一個基於 **Google Gemini API** 與 **Google Apps Script (GAS)** 開發的互動式網頁數學練習遊戲。學生將扮演拆彈專家，透過快速計算數學題來解除炸彈威脅，適合國小三至四年級學生練習。

## 🌟 遊戲特色

* **沉浸式情境**：科幻風格介面，搭配緊張的倒數計時與生命值系統。
* **動態 AI 出題**：利用 Gemini AI 根據關卡難度自動生成題目。
* **極簡化反饋**：答題階段 AI 僅回覆「正確/錯誤」，確保學生專注於計算。
* **固定式佈局**：優化 CSS 設計，確保計時器與狀態欄位在對話增加時不會位移或壓縮。

---

## 🎮 遊戲規則

1.  **挑戰目標**：連續正確解除 **5 顆** 數學炸彈即可獲勝。
2.  **防禦能量 (HP)**：初始擁有 **3 顆心**。答錯、超時或主動「跳過」都會扣除 1 顆心。
3.  **拆彈時限**：每題僅有 **60 秒**，倒數結束未完成視同拆彈失敗。
4.  **難度梯度**：
    * **Level 1-2**：四位數大數加減法。
    * **Level 3-4**：包含小括號的四則混合運算。
    * **Level 5**：單位換算應用題（公里與公尺）。

## 🚀 快速開始與部署

### 1. 取得 API Key
前往 [Google AI Studio](https://aistudio.google.com/) 建立並複製您的 API Key。

### 2. 設定 Google Apps Script (GAS)
1. 建立一個新的 GAS 專案。
2. 貼入 `Code.gs` 程式碼。
3. 點擊 **「專案設定 (齒輪圖示)」** > **「指令碼屬性」**。
4. 新增屬性：名稱為 `GEMINI_API_KEY`，值為您的 API Key。
5. 點擊 **「部署」** > **「新部署」**：
    * 類型：網頁應用程式
    * 誰可以存取：**所有人 (Anyone)**
6. 複製產生的 **網頁應用程式網址**。

### 3. 設定前端 HTML
1. 打開 `index.html`。
2. 找到 `const GAS_URL = '...';` 並換成您剛才複製的網址。
3. 使用瀏覽器打開 `index.html` 即可開始遊戲。

## ⚠️ 常見問題排除 (Troubleshooting)

* **連線異常**：請確認 GAS 部署時「誰可以存取」是否設定為「所有人」。
* **API 錯誤**：請檢查「指令碼屬性」中的 Key 是否正確，或 API 配額是否用盡。
* **顯示壓縮**：本版本已加入 `flex-shrink: 0`，若仍有跑版請檢查瀏覽器縮放比例。
