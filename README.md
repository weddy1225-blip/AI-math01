## 🎮 遊戲規則
1. **任務目標與計分系統**
基礎挑戰：連續通過 8 個任務 即可成功解除威脅，獲得基礎總分 100 分。
單題分值：前 8 題每題價值 12.5 分。
魔王紅利：如果你在 8 題中全對，將觸發隱藏的「魔王挑戰題」，成功拆除可額外獲得 50 分（最高總分可達 150 分）。

2. **防禦能量 (HP)**
初始生命：你擁有 3 顆心 (❤️❤️❤️)。
扣血機制：只要「答錯」或「60 秒倒數結束」，都會扣除 1 顆心，並自動跳至下一題。
任務失敗：若 3 顆心全部扣完，炸彈將會引爆，遊戲立即結束。

3. **拆彈時限**
黃金 60 秒：每一題只有 60 秒的時間思考與輸入答案。
緊急警報：當時間剩下最後 10 秒 時，計時器會變紅並開始震動，提醒你加快速度！

4. **任務難度梯度**
拆彈任務將依照下列順序出現，挑戰你的腦力極限：
Level 1-2：四位數大數加法與減法。
Level 3-4：四年級程度的四則混合運算（含小括號）。
Level 5-7：重量、長度、容量的單位換算與計算。
Level 8：面積計算應用題（長方形面積）。
Final Boss：高難度的進階混合運算邏輯。


## 🌟 遊戲特色

* **沉浸式情境**：科幻風格介面，搭配緊張的倒數計時與生命值系統。
* **極簡化反饋**：答題階段 AI 僅回覆「正確/錯誤」，確保學生專注於計算。
* **固定式佈局**：優化 CSS 設計，確保計時器與狀態欄位在對話增加時不會位移或壓縮。
